## 📖 Overview

The **Institute of Digital Risk (IDR)** is an industry-led training and deployment institute focused on digital, cyber, and technology risk roles in high-consequence sectors. This repository contains the source code for the IDR marketing and landing website.

The site bridges the gap between academic knowledge and real-world industry challenges — showcasing IDR's programs, research initiatives, and community for the next generation of risk professionals.

---

## ✨ Features

- ⚡ **Sticky Navbar** with scroll-based transitions and active link highlighting
- 🎞️ **Reveal Animations** using the Intersection Observer API for smooth on-scroll effects
- 📊 **Animated Stats Counter** — numbers count up when scrolled into view
- 📱 **Fully Responsive** — mobile menu overlay with a hamburger toggle
- 📬 **Contact Form** with simulated async feedback (loading → success states)
- 🎨 **Premium Design** — Inter font, custom CSS variables, and a consistent orange-on-dark brand palette
- 🔢 **Lucide Icons** for lightweight, consistent iconography

## 🚀 Getting Started

No build tools or dependencies required. This is a pure HTML/CSS/JS site.

### Run Locally

```bash
# Clone the repository
git clone https://github.com/your-org/idr-website.git

# Navigate into the project
cd idr-website

# Open in your browser
open index.html
```

Or serve it with any static file server:

```bash
# Using Python
python3 -m http.server 3000

# Using Node.js (npx)
npx serve .
```

Then visit `http://localhost:3000`.
